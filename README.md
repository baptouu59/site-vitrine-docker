# Projet Final DevOps – Site Vitrine Personnel Conteneurisé

## Étudiant
MAES Baptiste

## Sujet
Déploiement d’une application web conteneurisée avec Docker et Docker Compose

## Présentation du projet
Ce projet consiste à concevoir et déployer un **site vitrine personnel**
reposant sur une architecture web conteneurisée.
L’objectif est de mettre en pratique les principes de Docker et Docker Compose
à travers une application web simple mais structurée.

Le site vitrine permet de présenter l’étudiant, le projet, et de démontrer
la communication entre un frontend, une API backend et une base de données.

---

## Fonctionnalités
- Affichage d’un site vitrine personnel
- Récupération de données via une API backend
- Stockage des données dans une base de données
- Déploiement complet via Docker Compose

---

## Architecture technique
L’architecture de l’application est composée de :

- **Frontend Web**  
  Site vitrine développé en HTML / CSS / JavaScript

- **Backend API**  
  API REST simple permettant de fournir des données au frontend

- **Base de données**  
  Base de données relationnelle (MySQL ou PostgreSQL)

- **Docker & Docker Compose**  
  Chaque composant est conteneurisé et orchestré via Docker Compose
